# Searching & Finding Content

VIVIM makes it easy to find your past conversations. Whether you remember a specific detail or just want to explore, powerful search has you covered.

---

## Search Modes

### Local Search
- Searches conversations stored on your device
- Works offline
- Faster for smaller libraries

### Server Search  
- Searches all your conversations on the server
- Requires internet connection
- Searches entire library regardless of device

---

## How to Search

1. Tap the **Search** icon in the bottom navigation
2. Type your search query
3. Results appear automatically as you type (debounced 300ms)
4. Tap any result to open the conversation

---

## What You Can Search By

### Exact Matches
- Conversation titles
- Provider names (chatgpt, claude, gemini)

### Semantic Search (Server)
- Content meaning (even if keywords don't match)
- Topics and themes
- Concepts discussed

---

## Search Tips

### Finding Specific Conversations
- Search for project names
- Search for people mentioned
- Search for code snippets

### Example Searches

| Query | Finds |
|-------|-------|
| "React hooks" | Conversations about React hooks |
| "API design" | API design discussions |
| "Claude" | Conversations with Claude |
| "error handling" | Error handling patterns |

---

## Filters & Organization

After finding results, you can:

- **Pin** important conversations
- **Archive** old ones
- **Add to Collections** for organization
- **Bookmark** for quick access

---

## Advanced Search

For more advanced searching, visit **Context Components** (`/context-components`) to:
- Browse ACUs (Active Context Units)
- View- Explore your knowledge graph

---

## context bundles
 Troubleshooting

### No Results?
- Try both Local and Server search
- Use fewer/more generic keywords
- Check spelling

### Slow Search?
- Local search is faster for small libraries
- Server search may be slower with large libraries
- Consider organizing with Collections
