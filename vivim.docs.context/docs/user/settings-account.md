# Settings & Account

Manage your VIVIM account, preferences, and personal information.

---

## Account Section

### Signing In
- **Google Sign-In** - Connect your Google account
- Benefits: Cloud sync, cross-device access, backup

### Account Info
When signed in, you see:
- Email address
- Display name
- Account status (Active/Pending Deletion)
- Verification level
- Trust score

### DID (Decentralized Identifier)
Your unique VIVIM identity:
- Starts with `did:vivim:...`
- Used for sharing and verification
- Shown in Settings when not signed in with Google

---

## Verification Levels

VIVIM tracks verification depth:

| Level | Verification | Badge |
|-------|--------------|-------|
| 0 | None | None |
| 1 | Email verified | ✓ |
| 2 | Phone verified | ✓✓ |
| 3 | Identity verified | ✓✓✓ |

Higher levels = more trust in the network.

---

## Account Deletion

### Request Deletion
1. Go to **Account** page
2. Scroll to Danger Zone
3. Tap **Delete Account**
4. Confirm with email

### What Happens
- 30-day cancellation window
- Data export available
- All data permanently deleted after

### Cancel Deletion
If you change your mind:
1. Go to **Account** within 30 days
2. Tap **Cancel Deletion**
3. Account restored

## App Settings

### Theme
- **Dark Mode** - Toggle dark/light theme
- Follows system preference by default

### Notifications
- **Enable/Disable** - Toggle push notifications

### Sync
- Data syncs automatically when signed in
- Manual sync available in Settings

---

## Data Management

### Clear Local Data
1. Go to **Settings**
2. Find data section
3. Tap **Clear Data**
4. Confirm

This removes local data only. Cloud data remains if signed in.

### Export Data
Request data export:
1. Account settings
2. Tap **Export Data**
3. Receive download link via email

---

## Device Settings

### Device Type
VIVIM adapts to your device:
- **Auto** - Detected automatically
- **Force Desktop/Mobile** - Override detection
- **Reset** - Return to auto detection
