## Design Notes

1. **Light Mode Palette**:
   - Background: `#FAFAFA` (soft white)
   - Cards: `#FFFFFF` with subtle shadow
   - Primary accent: `#6366F1` (indigo) to `#8B5CF6` (violet)
   - Text: `#1A1A2E` (near black) for headlines, `#6B7280` (gray) for secondary
   
2. **Spacing**: Generous whitespace, never crowded
   
3. **Buttons**: Single-action focus, one primary CTA per screen
   
4. **Key Concepts to Visualize**:
   - **ACU**: Atomic Chat Units as modular, shareable knowledge blocks
   - **VaultSense**: Personal knowledge graph/constellation
   - **Collaboration**: Fork trees, contributor avatars, community building
