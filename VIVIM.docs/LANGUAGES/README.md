## Quick Reference

### Core Principles
1. **"Know the source, read in yours"** - Users always see source language
2. **Quality tiers** - Machine → Community → Professional → Official
3. **User preference-driven** - Auto-translate based on language knowledge graph
4. **Rich content support** - Markdown, code blocks, media preserved

### Key Concepts
- **Source Language** - Original language of the content
- **Reading Language** - User's preferred language
- **Quality Indicator** - Trust level of translation
- **Bilingual View** - Side-by-side or inline comparison
