# Contributing to VIVIM SDK

We're excited that you're interested in contributing to the VIVIM SDK! 

## ðŸ›  Development Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vivim-app/vivim-app.git
   cd vivim-app/sdk
   ```

2. **Install dependencies:**
   We recommend using [Bun](https://bun.sh/) for development.
   ```bash
   bun install
   ```

3. **Running Tests:**
   Ensure everything is working correctly:
   ```bash
   bun run test
   ```

4. **Building:**
   ```bash
   bun run build
   ```
